windows中如何查看代理的地址和端口

查看状态

git config –global -l

给git设置代理,这里10809是本地端口号，查找自己本地的电脑端口号

git config –global http.proxy 127.0.0.1:10809
git config –global https.proxy 127.0.0.1:10809
